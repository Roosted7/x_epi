Metabolite Specific#

x_epi: An EPI sequence for X-Nucleus imaging

usage:  [-h] [-name NAME] [-grd_path PATH] [-rf_path PATH] [-formula FORMULA] [-use_sinc] [-flip ANGLE] [-freq_off OFFSET] [-sinc_dur DUR] [-sinc_frac FRAC] [-sinc_tbw TBW] [-size RO PE SLC]
        [-pf_pe FRAC] [-pf_pe2 FRAC] [-z_centric]

Named Arguments#

-name

Metabolite name

Metabolite Specific Waveforms#

-grd_path

Path to gradient file from Spectral-Spatial RF Toolbox

Default: “/Users/blazeyt/Documents/code/x_epi/x_epi/res/siemens_singleband_pyr_3T.GRD”

-rf_path

Path to RF file from Spectral-Spatial RF Toolbox

Default: “/Users/blazeyt/Documents/code/x_epi/x_epi/res/siemens_singleband_pyr_3T.RF”

-formula

Formula for scaling SSRF gradient where x is slice thickness in mm

Default: “1”

Metabolite Specific Excitation#

-use_sinc

Use sinc pulse instead of an ssrf pulse

Default: False

-flip

Flip angle [deg]

Default: 90

-freq_off

Frequency offset [Hz]

Default: 0

-sinc_dur

Duration of sinc pulse [ms]

Default: 4

-sinc_frac

Center fraction of sinc pulse

Default: 0.5

-sinc_tbw

Time-bandwidth product of sinc pulse

Default: 4

Metabolite Specific Imaging Parameters#

-size

Grid size in readout, phase, and slice dimensions

Default: [16, 16, 16]

-pf_pe

Partial Fourier in phase encoding dimension

Default: 1

-pf_pe2

Partial Fourier in second phase encoding dimension

Default: 1

-z_centric

Use centric encoding instead of linear for second phase encoding dimension.

Default: False

Each metabolite must be preceded by -met.
